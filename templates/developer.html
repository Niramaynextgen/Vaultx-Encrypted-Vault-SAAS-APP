About VaultX

VaultX is a SaaS-style password vault web application built using Django. It allows users to securely store, manage, and retrieve login credentials for various websites and services. All sensitive data is encrypted using Fernet symmetric encryption (AES-128-CBC + HMAC-SHA256) before being stored in the database — meaning even if the database is compromised, passwords remain unreadable without the secret key.

This project was built as part of the project topic: "Secure Login and Authentication by Cryptography."
